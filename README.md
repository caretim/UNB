## 0. 개발 목적

그동안 Django만을 사용해 풀스택 개발을 해온 팀원들끼리 모여, 백엔드와 프론트엔드 환경을 완전히 분리해 RESTFUL API 개발 경험을 얻기 위해 시작했습니다.

## 1. 서비스 소개

UNB는 다양한 주제들로 양자택일 토론을 하고, 가볍게 즐길 수 있는 사이트입니다. 또한 뱃지와 등급제도를 통해 한 번 사용하고 마는 사이트가 아닌, 사용자들이 지속적으로 즐길 수 있는 서비스입니다.
